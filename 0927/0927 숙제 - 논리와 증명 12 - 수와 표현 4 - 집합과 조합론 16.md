논리와 증명 12 

```
대우 "자연수 n에 대하여 n이 3의 배수가 아니면

n²도 3의 배수가 아니다"

임의의 자연수 n은..

n=3k

n=3k -1

n=3k -2

셋 중 하나가 됩니다. (단 k는 자연수) 

n이 3의 배수가 아닌경우는

n=3k를  제외한 2가지로

i) n=3k-1 인 경우 

n²=(3k-1)²=9k²-6k+1 = 3(3k²-2k) +1

로 3으로 나누면 나머지가 1로 3의배수가 아닙니다.

ii) n=3k-2 인 경우

n²=(3k-2)²=9k²-12k+4 =3(3k²-4k+1) +1

로 3으로 나누면 나머지가 1로 역시 3의 배수가

아닙니다.

n이 3의 배수가 아닌 모든 경우에서

n² 도 3의 배수가 아니기에 대우가 참이 되어

원래 명제도 참이 됩니다.
```

수와 조건 4 

```
문제 4 : x = loga(yz) 일때 x를 2를 밑으로 하는 로그들로 표현하시오. 단, 로그 함수의 인자는 모두 문자 하나여야 한다.

풀이 : 

loga(yz) = loga(y) + loga(z)

= log2(y)/log2(a) + log2(z)/log2(a)

= (log2(y)+log2(z)) / log2(a)
```
